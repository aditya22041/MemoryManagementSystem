MeMS‑Pro: Advanced Memory Manager

High‑performance, thread‑safe slab & page allocator in C++17

Overview

MeMS‑Pro is a C++ library that simulates OS‑style memory management:

  • Slab Allocator for objects ≤512 B
    (mmap-based)
  • Paged Allocator for larger blocks via mmap(2)
  • Select First‑Fit or Best‑Fit strategies
  • Thread-safety with pthread_mutex_lock(3p)
  • Benchmarked workloads & Python visualization
  • Unit tests via GoogleTest

API Example

#include "memory_manager.h"

int main() {
    MemoryManager mem(1024*1024, Strategy::BEST_FIT);
    int* arr = static_cast(mem.alloc(1000 * sizeof(int)));
    if (!arr) return -1;
    for(int i=0;i<1000;++i) arr[i] = i*i;
    mem.free(arr);
    return 0;
}
