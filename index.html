MeMS‑Pro: Advanced Memory Manager

High‑performance, thread‑safe slab & page allocator in C++17

Project Overview

MeMS‑Pro is a C++ library that mimics OS‑level memory management:

  • A Slab Allocator for small, fixed-size objects (≤512B).
  • A Paged Allocator via mmap() for larger blocks.
  • Selectable First‑Fit or Best‑Fit strategies.
  • Fully thread‑safe using pthread_mutex_lock(3p).
  • Benchmarked workloads with CSV output and Python plots.
  • Unit‑tested via GoogleTest.

API Usage Example

#include "memory_manager.h"

int main() {
    // 1 MB pool, best‑fit
    MemoryManager mem(1024*1024, Strategy::BEST_FIT);

    // Allocate an int array of 1000 elements
    int* arr = static_cast(mem.alloc(1000 * sizeof(int)));
    if (!arr) return -1;

    for(int i=0;i<1000;++i) arr[i] = i*i;
    mem.free(arr);
    return 0;
}

See std::alloc docs and mmap(2) man page for reference.
